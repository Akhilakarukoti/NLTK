## 🔧 Features

- ✍️ **Text Summarizer**  
  Upload or paste text and generate a summary using NLTK.

- 🎤 **Voice to Text**  
  Speak into your microphone to get instant transcription.

- 🔊 **Text to Voice**  
  Convert written content into human-like speech.
